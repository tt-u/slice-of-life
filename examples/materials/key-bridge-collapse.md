# 巴尔的摩 Francis Scott Key Bridge 大桥坍塌（2024）

## 背景
2024 年 3 月，货船 Dali 撞击巴尔的摩 Francis Scott Key Bridge，导致桥梁坍塌。事故涉及船东、运营方、港口、州政府、联邦机构、保险方、物流企业和公众，迅速演化为基础设施、航运、救援与责任认定的综合危机。

## 关键事实
- 事故造成重大基础设施损毁与人员伤亡，并中断重要港口航道和区域物流。
- 早期问题集中在船舶电力故障、应急响应、航道管理和维护责任。
- 后续重点包括搜救、清障、航道恢复、港口复工和赔偿责任。
- 物流、就业、燃料、汽车运输和地方政治都受到影响。
- 不同主体掌握的信息不完整，恢复目标也不一致。
- 舆论可能在“不可抗意外”“可预防维护失误”“基础设施脆弱性”之间摇摆。

## 可提炼实体
- 船东 / 船舶管理方
- 港口管理者
- 州政府 / 联邦机构
- 保险与法律团队
- 物流客户 / 港口劳工
- 媒体与地方公众

## 潜在玩家视角
- 港口危机协调负责人
- 州政府应急负责人
- 航运集团风险负责人

## 核心冲突
- 优先救援、清障、复航还是先稳定政治和公众叙事？
- 如何在多机构、多责任主体环境下维持控制权？
- 基础设施与商业恢复的节奏由谁决定？
